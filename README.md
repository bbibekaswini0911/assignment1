# Pikachu Card Website

A simple and responsive Pikachu themed card website created using HTML and CSS.

## Features
- Clean UI Design
- Responsive Layout
- Hover Button Effect
- Pikachu Image Card

## Technologies Used
- HTML5
- CSS3

## Preview
This project displays a stylish Pikachu information card with modern design.

## How to Run
1. Download the project
2. Open `assignment1.html`
3. Run in browser
